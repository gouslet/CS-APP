2.55 在你能够访问的不同机器上，使用show_bytes（文件show-bytes.c)编译并运行示例代码。确定这些机器使用的字节顺序。

|机型| 字节顺序|
|---|---|
|华硕X550L|小端序|

    #include "show_bytes.h"

    int main(void)
    {
        int a = 12;

        show_bytes((byte_pointer)&a, sizeof(int));
    }

    /** 
        * show_bytes 
        */
    void show_bytes(byte_pointer start, size_t len)
    {
        size_t i;
        for (i = 0; i < len; i++)
        {
            printf("%.2x", start[i]);
        }
        printf("\n");
    }

运行以上程序可得如下结果：

    0c000000
    00004041
    ccfd660000000000

$0c = 12$位于低地址，可见该机器为小端序
