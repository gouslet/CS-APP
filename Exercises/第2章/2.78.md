2.78 写出具有如下原型的函数的代码：
```c
/* Divide by power of 2. Assume O <= k < w-1 */
int divide_power2(int x, int k);
```
该函数要用正确的舍入方式计算$x/2^k$, 并且应该遵循位级整数编码规则。  
解：
- 思路：判断符号位，然后对负数做修正
- 代码：
```c
int divide_power2(int x, int k) {
    int sign = INT_MIN & x;
    sign && x += (1 << k) - 1;
    return x >> k;
}

```
