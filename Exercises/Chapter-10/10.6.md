10.6 下面程序的输出是什么？
```c
#include "csapp.h"

int main() {
    int fd1, fd2;
    fd1 = Open("foo.txt", O_RDONLY, 0);
    fd2 = Open("bar.txt", O_RDONLY, 0);
    Close(fd2);
    

    fd2 = Open ("baz.txt", O_RDONLY, 0);
    printf("fd2 = %d\n", fd2);
    exit(0);
}
```
- 猜想：
  `fd1 = 3`，`fd2 = 4`，`Close(fd2)`执行之后，`fd2`被释放，`fd2 = Open ("baz.txt", O_RDONLY, 0)`执行之后，`fd2=4`
- 验证：
    在centos上运行本程序，结果如下
```
    [root@hecs-275609 src]# ./10.6
    fd2 = 4
```
