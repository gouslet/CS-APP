2.30 写出一个具有如下原型的函数：

    /* Determine whether arguments can be added without overflow */
    int tadd_ok(int x, int y);
如果参数x和y相加不会产生溢出，这个函数就返回1。

    int tadd_ok(int x,int y){
        int z = x + y;
        if (!(x >= 0 && y >= 0 && z < 0) && !(x < 0 && y < 0 && z >= 0))
            return 1;
    }
