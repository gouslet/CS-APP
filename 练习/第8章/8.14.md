下面这个程序会输出多少个“hello" 输出行？
```c
#include "csapp.h"
void doit () {
    if (Fork() == 0) {
        Fork();
        printf ("hello\n");
        exit(0);
    }
    return;
}
int main() {
    doit();
    printf ("hello\n");
    exit(0);
}
```
解：5行  
第一次Fork之后父进程输出一行hello，子进程再次Fork，各输出2行hello，共5行  
在华为云contos上运行，仅仅输出1行
