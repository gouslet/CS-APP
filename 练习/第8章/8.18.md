8.18 考虑下面的程序：
```c
#include "csapp.h"

void end(void) {
    printf("2");
    fflush(stdout);
}

int main() {
    if(Fork() == 0)
        atexit(end);
    if(Fork() == 0) {
        printf("0");
        fflush(stdout);
    } else {
        printf("1");
        fflush(stdout);
    }
    exit(0);
}
```
判断下面哪个输出是可能的。注意：atexit函数以一个指向函数的指针为输入，并将它添加到函数列表中（初始为空），当exit函数被调用时，会调用该列表中的函数。
