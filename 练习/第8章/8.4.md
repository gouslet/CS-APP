8.4 考虑下面的程序：
```c
int main() {
    int status;
    pid_t pid;
    
    printf("Hello\n");
    pid = Fork();
    printf("%d\n", !pid);

    if(pid!=0) {
        if (waitpid(-1, &status, 0) > 0) {
            if (WIFEXITED(status) != 0)
                printf("%d\n", WEXITSTATUS(status));
        }
    }
    printf("Bye\n");
    exit(2);
}
```
A. 这个程序会产生多少输出行？
    4+2=6行
B. 这些输出行的一种可能的顺序是什么？
```
Hello
0
1
Bye
2
Bye
```
